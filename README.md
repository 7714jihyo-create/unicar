## UniCar 유니티 자동차 레이싱 게임

유니티(Unity) 2D 환경에서 개발한 미니카 레이싱 게임입니다. 키보드의 좌우 화살표 버튼으로 자동차를 이동시킬 수 있으며, 초당 1점씩 점수가 증가합니다.
다른 자동차와 충돌할 시 충돌 효과음과 함께 게임이 종료됩니다.
